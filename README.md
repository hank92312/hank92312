## 📊 精選專案 | Featured Projects

### 🎬 [AI 自動化短影音內容流水線](https://github.com/hank92312/ai-content-pipeline)

運用 Python 串接大語言模型 API 打造的自動化內容生成系統。全程使用 Claude Code 與 GPT-Codex 協作開發，優化提示詞工程（Prompt Engineering），將影音企劃、腳本生成至多平台素材對接流程完全數位流水線化，大幅縮短內容產出時間。

### 📈 [Savestock — 長線存股防護系統](https://github.com/hank92312/Savestock)

專為長線投資與資產防護設計的財務大數據分析系統。利用 Google Gemini (Antigravity) 輔助編寫複雜的 SQL 預存程序與 Pandas 資料預處理腳本，自動清洗並爬取台股歷史配息與投報率數據，將大數據（EDA）視覺化呈現，用客觀數據輔助資產防護與決策。

### 🎲 [Friends&Me — 異地社交探索桌遊](https://github.com/hank92312/Friend-Me)

展現全端開發邏輯的社交/媒合應用專案。前端使用 HTML5 與 Bootstrap 打造直覺的 UI/UX 界面，後端以 C# 處理核心商業邏輯，並串接 MS SQL Server 進行高效能的資料庫設計與權限管理，開發期間善用 AI 工具進行程式碼重構與單元測試自動生成。

### 📝 [HackMD 筆記批次下載與上傳工具](https://github.com/hank92312/HackMDbackup)

為解決數位資產備份痛點而開發的 Python 自動化批次下載工具，利用 Python 腳本串接 API 與自動化流程，高效、精準地將 HackMD 上的筆記進行批次封裝與本地端備份。
